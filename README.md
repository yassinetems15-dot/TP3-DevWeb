# TP3 – Application Web Interactive avec JavaScript

## Description
Enrichissement d'une page HTML/CSS en application web dynamique
grâce à JavaScript natif.

## Module
Développement Web 1 – CI ISI  
Pr. Abdelouahed Tricha | ISGA Casablanca | 2025/26

## Fonctionnalités
- Validation dynamique du formulaire
- Ajout et suppression de modules (DOM)
- Mode sombre / clair
- Chargement d'utilisateurs via AJAX (fetch)
- Affichage de l'email au clic (bonus)

## Technologies utilisées
- HTML5
- CSS3
- JavaScript natif 

## Structure des fichiers
- index.html → structure de la page
- style.css  → mise en forme + dark mode
- script.js  → toutes les fonctionnalités JS

## Lancer le projet
Ouvrir index.html dans un navigateur
